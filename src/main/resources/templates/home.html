Project Implimentations:

Implimentation
Description
CRUD
Can perform CRUD Operations (ie. CREATE, RETRIVE, UPDATE, DELETE).
Security
Authenticated Data Access with proper Authentication implemented with Spring Security Protocols.
oAuth2 Authentication
External libraries (like Google, Github) is implemented for genuine authentication of Users.
Hibernate (JPA)
An ORM(Object Relational Mapping) tool based on JPA Protocol, is implemented for mapping Java data types with SQL data types and database tables.
Database
MySQL Database is implemented to store data on the server.
Media Storege Service
To Store media, Cloudinary API is integrated in order to store and fetch media (like Images, logo etc).
Mail Service
To Send mails (including e-mail verifications), Mailtrap & GMail API's are integrated for seamless Interaction & Authentication.
